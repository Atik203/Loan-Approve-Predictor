# Loan Approve Predictor

This project is a machine learning model that predicts whether a loan application will be approved or not based on various features of the applicant and the loan. The dataset contains information such as the applicant's income, credit history, loan amount, and more. The goal is to build a model that can accurately classify loan applications as approved or not approved, which can help financial institutions make informed decisions about lending.

**Link to dataset**: [Loan Prediction Dataset](https://www.kaggle.com/datasets/ninzaami/loan-predication)

**Demo Link**: [Loan Approve Predictor](https://huggingface.co/spaces/atik203/Loan-Approve-Predictor)
